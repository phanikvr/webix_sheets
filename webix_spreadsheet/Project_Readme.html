This project is built in VS2015 with .net framework 4.6

  • pre-requisities:
  • VS 2015
  • .Net framework 4.6 or above

Brief about the project :

  • this .net app is created for API to save data to DB and hosted the html files for the sake of avoiding CORS.(we can fix by enabling cors at server side and modifying webix api to handle preflight requests.)
  • a LOCAL SQL MDF file is selected for DB operations and it is included in the project.
  • webix client is included in the project under scripts/codebase folder.
  • no other UI frameworks are used except WEBIX.
  • the project is set to open readme.html file instead of default serverpage. if it goes to default page please redirect to index.html.
  • after clicking the save button, we need to wait until the green "tick" image loads before going to next page.
  • lot of packages are loaded by default which are not required. need to clean up.

Assumptions:

  • no proper Schema is defined for EXCEL, So 4 columns have been considered and if not provided, automatically adding the columns.
  • last updated records to be available on top for editing.
  • Only one row must be edited at any given time.
